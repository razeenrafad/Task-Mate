Create a Flask web app named Daily Task Tracker.
Features:

Add, edit, delete tasks

Mark as completed

Store tasks using SQLite database

Use HTML template for the front-end

Add a CSS file for basic styling

Use the following structure:
app.py
templates/index.html
static/style.css
requirements.txt
Procfile
README.md
Make sure:

app.py includes SQLAlchemy for database

index.html shows tasks dynamically

style.css is simple and clean

README explains how to run locally and deploy on Render
